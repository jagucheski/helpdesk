# HelpDesk

## About
A friendly web application for ticket management of IT support.

## Motivation
Manage and track service calls from IT support with exchange between operator and requester.

## Technologies

- Spring MVC;
- Spring Security;
- Hibernate;
- Postgresql;
- Tomcat;
- Jquery.
